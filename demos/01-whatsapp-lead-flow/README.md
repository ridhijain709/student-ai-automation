# WhatsApp Lead Flow

This demo showcases how to automate lead flow through WhatsApp. It demonstrates the entire journey of a lead from initiation to follow-up.

## Features
- Automated message sending
- Lead status tracking

## Setup
1. Clone the repository.
2. Configure the WhatsApp API.
3. Run the automation script.

## Usage
- Just start the bot and communicate with the leads via WhatsApp!
